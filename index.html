Acerca de este sitio

Este sitio web ha sido desarrollado como parte de un proyecto universitario para la materia de Desarrollo Web de la Licenciatura en Informática y Tecnologías Computacionales.

El objetivo es presentar información detallada sobre tecnologías web modernas, incluyendo Web APIs, Microservicios, Desarrollo de Web APIs, Manejo de Contenedores y Desarrollo de Aplicaciones para la Nube.

Cada sección contiene información teórica, ejemplos prácticos y recursos para profundizar en los temas.
